# Carter's (carters)

Carter's, Inc. is the largest branded marketer of young children's apparel in the United States, operating the Carter's, OshKosh B'gosh, and Skip Hop brands. Its customer-facing digital surface includes carters.com and carters.com/cart e-commerce, mobile apps, a rewards program, and vendor / supply-chain integrations, but no public developer API or partner program is currently published.

## APIs

No public developer APIs have been documented for Carter's at this time. Carter's digital integration surface is focused on:

- Direct-to-consumer e-commerce (carters.com) with a Rewarding Moments loyalty program and mobile apps.
- Brand-site properties for OshKosh B'gosh and Skip Hop.
- Internal Order Management System (OMS) and supply-chain integrations supporting stores, wholesale partners, and distribution centers.

If a public affiliate, product-catalog, or inventory API is published in the future, contributions are welcome.

## Use Cases

- Direct-to-consumer shopping on carters.com with loyalty redemption through Rewarding Moments.
- Multi-brand browsing across Carter's, OshKosh B'gosh, and Skip Hop.
- Wholesale and department-store inventory replenishment via Carter's internal OMS and EDI integrations (not publicly exposed).
- Investor and analyst workflows via the IR site.
